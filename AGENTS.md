# AGENTS.md

Guidance for human and AI contributors working in this repository.

## Purpose

This repo is the public Winter Advisory website and intake surface.
It is not the canonical Winter Advisory operating system.

If you need to coordinate company work, review portfolio priorities, inspect
operating briefs, or understand what Winter Advisory is doing today, use the
real HQ first:

- Canonical HQ repo: `/Users/andrewwinter/Documents/GitHub/winter/hq`
- Winter Advisory instance: `/Users/andrewwinter/Documents/GitHub/winter/hq/instances/winter-advisory`

## Repo Boundary

Work that belongs here:

- marketing site copy, design, and UX
- lead capture and intake flows
- public positioning and contact surfaces
- analytics related to the public site

Work that does not belong here:

- portfolio operating decisions
- company scorecards, routines, and reports
- product-repo prioritization across the portfolio
- shared agent operating instructions for Winter Advisory as a whole

## Agent Routing Rule

When a request mentions any of the following, leave this repo and start from the
HQ instance path above unless the user explicitly asks for website work:

- Winter Advisory HQ
- portfolio coordination
- what to work on today
- company priorities
- operating reports
- scorecards
- delegation across businesses

If the request is specifically about the website, intake widget, contact flow,
or public positioning, stay in this repo.
